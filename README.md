# CPP-STL-Applications
Contains applications written in the C++ Programming Language that are compatible with the Standard Template Library (STL). 
The applications are related to topics such as object-oriented programming, TCP communication, serial port communication, file operations, and the use of threads.

Programming Language: C++ Programming Language

Programming Language Version: STL C++11

Framework: Qt5 (Version: Qt 5.11.3)

IDE: Qt Creator 4.8.2
